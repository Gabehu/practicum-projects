The `customer_information` table (data on customers):

- *Unnamed: 0* - Index
- *id*
- *first_name*

- *customer_gender*
- *email*
- *sign_up_date*
- *last_log_date*
- *country*
- *city*
- *Industry* - Type of industry
- *Size* - Size of company
- *Goal* - Customer goal
- *Role* - Customer Role

- *sms_purchase_date*
- *sms_requested*
- *total_cost_usd*
